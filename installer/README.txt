# Localization Comparison Tool

A powerful desktop application for comparing, analyzing, and managing localization files.

## What's New in v2.0.0

### Major UI/UX Overhaul
- Side-by-side diff viewer with synchronized scrolling
- Character-level diff highlighting showing exact changes
- Filter badges for Added/Removed/Modified with click-to-toggle
- Reviewed row visual feedback for tracking progress
- Rich empty states with helpful guidance

### Git Comparison Improvements
- Enhanced visual hierarchy with branch/repo icons
- Prominent file status indicators with color-coded strips
- Consistent styling across all comparison views

### Settings & Polish
- Collapsible sections for better organization
- Contextual tooltips explaining each setting
- Status indicators for backup and protocol registration

## Features

- **Multi-Format Support**: JSON, ARB, XML, XLIFF, TMX, CSV, YAML, Properties, RESX
- **Directory Comparison**: Compare entire folders at once
- **Git Integration**: Compare branches and commits directly
- **Quality Dashboard**: Analytics and translation progress tracking
- **AI Translation**: Integration with Gemini, OpenAI, Google Translate, DeepL
- **History Tracking**: Automatically saves all comparison sessions
- **Modern Theming**: Light, Dark, and AMOLED modes

## System Requirements

- Windows 10 or later (64-bit)
- 100 MB disk space
- 4 GB RAM recommended

## Getting Started

1. Launch the application
2. Drag and drop localization files to compare
3. Use the Advanced Diff view for detailed analysis
4. Export results or save to history

For more information, visit:
